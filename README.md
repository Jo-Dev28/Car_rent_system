# 🚗 Car Rental Management System

A modern web-based Car Rental Management System developed to simplify vehicle booking, rental management, customer registration, and administrative operations. The system provides an efficient platform for customers to browse available vehicles, make reservations, and manage rentals online while allowing administrators to manage vehicles, bookings, users, and payments.

## 📋 Features

### Customer Features
- User Registration and Login
- Browse Available Vehicles
- Search and Filter Cars
- Vehicle Details Page
- Online Car Booking
- Booking History
- Profile Management
- Contact Support

### Admin Features
- Secure Admin Dashboard
- Manage Vehicles
- Add, Edit, and Delete Cars
- Manage Customers
- Manage Reservations
- View Rental Reports
- Manage Payments
- Monitor System Activities

### Additional Features
- Responsive Design
- Real-Time Availability Checking
- Booking Status Tracking
- Image Upload for Vehicles
- Secure Authentication
- Notification System

## 📂 System Modules

### 1. User Management
Handles customer registration, login, profile updates, and account management.

### 2. Vehicle Management
Allows administrators to add, update, delete, and manage vehicle information.

### 3. Booking Management
Manages reservations, rental schedules, booking approvals, and cancellations.

### 4. Payment Management
Handles payment tracking and transaction records.

### 5. Reporting Module
Generates reports on bookings, customers, and vehicle utilization.
